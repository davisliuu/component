 300，或其他值。各个模块的频点的具体配置请参考“Hi3519V101专业型HD IP Camera SoC用户指南”。

/************************************************************************************************************/
 如果使用VI-VPSS在线模式，请将ISP和VPSS的时钟配置成一致。
/************************************************************************************************************/

以下sensor库目前仅在HI3519V101上提供：
/************************************************************************************************************/
IMX226 :
	使用4K*2k 2to1_frame WDR,VI 帧率达到60帧时，需要修改VI与ISP时钟为600MHz。
	使用4K*3k 2to1_frame WDR,VI 帧率达到30帧时，需要修改VI与ISP时钟为500MHz及以上。
	使用4K*3k@30，帧率达到30帧时，需要修改VI与ISP时钟为500MHz及以上。
	将mpp_xxx/ko/clkcfg_hi3519v101.sh中"himm 0x1201004c 0x00094821，himm 0x12010054 0x4041"修改为相应的值。
/************************************************************************************************************/

/************************************************************************************************************/
IMX274 :
	使用4K*2k 2to1_frame WDR,VI 帧率达到60帧时，需要修改VI时钟为600M,ISP时钟要达到300M及以上。
	将mpp_xxx/ko/clkcfg_hi3519v101.sh中"himm 0x1201004c 0x00094821，himm 0x12010054 0x4041"修改为相应的值。
/************************************************************************************************************/

/************************************************************************************************************/
IMX290 :
	使用1080P@30 2to1_line WDR,VI 帧率达到60帧时，需要修改VI时钟为300MHz及以上，ISP时钟214M及以上。
	使用1080P@30 3to1_line WDR,VI 帧率达到90帧时，需要修改VI时钟为340MHz及以上，ISP时钟214M及以上。
	将mpp_xxx/ko/clkcfg_hi3519v101.sh中"himm 0x1201004c 0x00094821，himm 0x12010054 0x4041"修改为相应的值。
/************************************************************************************************************/


以下sensor库目前仅在HI3559V100上提供：
/************************************************************************************************************/
IMX377 :
	使用1080P@120fps、720P@240fps、4k*2K@60或12M@30时，需要修改VI与ISP时钟为600MHz以上。
	将mpp_xxx/ko/clkcfg_hi3519v101.sh中"himm 0x1201004c 0x00094821"修改为相应的值。
/************************************************************************************************************/

mn34120 :

	使用16M@16.25时，需要修改VI与ISP时钟为300MHz及以上
	将mpp_xxx/ko/clkcfg_hi3519v101.sh中"himm 0x1201004c 0x00094821，himm 0x12010054 0x4041"修改为相应的值。
/************************************************************************************************************/

/************************************************************************************************************/
imx117 :
使用4K@30fps时，VI与ISP时钟最低需要300MHz。
使用4K@60fps时，VI与ISP时钟最低需要600MHz。
使用12M@28fps时，VI与ISP时钟最低需要396MHz。
将mpp_xxx/ko/clkcfg_hi3519v101.sh中"himm 0x1201004c 0x00094821，himm 0x12010054 0x4041"修改为相应的值。
